##JAVA##

**SCANNER**
```bash
import java.util.Scanner;
```

SIEMPRE que pones un SCANNER tienes que poner:
```bash
Scanner scanner = new Scanner(System.in);
```

Para poder pedir al usuario un dato para el programa.
EJEMPLO
```bash
System.out.print("Ingrese tu peso: ");
peso = scanner.nextDouble();
```

VARIABLES

las variables son los datos del programa

int: Representa un número entero sin decimales como 25 o -10

double: Representa un número con decimales como 3.14 o -5.67

String: Representa una secuencia de caracteres o texto como "Hola" o "Juan"

boolean: Representa un valor verdadero o falso como true o false
